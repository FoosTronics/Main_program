\doxysubsection{Detailed Description}
De basis van het hoofd testbed geraamte. 

If you are planning on using the testbed framework and\+:
\begin{DoxyItemize}
\item Want to implement your own renderer (other than Pygame, etc.)\+: You should derive your class from this one to implement your own tests. See empty.\+py or any of the other tests for more information.
\item Do N\+OT want to implement your own renderer\+: You should derive your class from \mbox{\hyperlink{namespacesrc_1_1_backend_1_1_framework}{Framework}}. The renderer chosen in fw\+Settings (see \mbox{\hyperlink{_settings_8py}{settings.\+py}}) or on the command line will automatically be used for your test. Als u van plan bent het testbed-\/framework te gebruiken en\+:
\begin{DoxyItemize}
\item U wilt uw eigen renderer implementeren (anders dan Pygame, etc.)\+: Leid hieruit je eigen klas om je eigen testen te implementeren. Zie empty.\+py of een van de andere tests voor meer informatie.
\item Wilt u uw eigen renderer N\+I\+ET implementeren\+: Leid je klas uit \mbox{\hyperlink{namespacesrc_1_1_backend_1_1_framework}{Framework}}. De gekozen renderer in fw\+Settings (zie \mbox{\hyperlink{_settings_8py}{settings.\+py}}) of op de opdrachtregel wordt automatisch gebruikt voor uw test. 
\end{DoxyItemize}
\end{DoxyItemize}
